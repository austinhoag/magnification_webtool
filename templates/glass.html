GLASS Cluster Magnification Webtool

Upload a file containing line(s) of the format:

Cluster, ID, RA, DEC, z (no spaces allowed)

E.g. :

# Header
MACS1149,593,177.406921,22.407505,1.48
MACS0416,94,64.033089,-24.056322,1.366
...
# end of file

Note 1: Lines beginning with '#' will be ignored
Note 2: Acceptable cluster names: A2744, MACS0416, MACS0717, MACS1149, RXJ2248, A370, MACS2129, MACS0744, MACS1423, RXJ1347
Note 3: The latest version of the lensing maps will be used automatically.
Note 4: If the magnification returned is -99, then the object is outside of the lensing FOV
Note 5: The model for MACS0744 is under construction, so currently magnification will be returned as -99 for all coordinates in that cluster
